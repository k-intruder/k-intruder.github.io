05. SMART 교육방법 배열

smart_methods[]
ℹ️ method_name은 아래 SMART 분류 중 하나를 선택합니다. 툴에서는 드롭다운으로 제공됩니다.
S
Sync (동기화 학습)
블렌디드러닝 · 플립드러닝 · 소셜러닝
M
Mirroring (현장 반영)
캡스톤디자인 · 산업체협업교육 · 사례기반학습 · 현장학습
A
Affiliation (협업)
협동학습 · 팀기반학습(TBL) · 하브루타학습 · 토론학습
R
Reinforcement (강화)
문제중심학습(PBL) · 자기주도학습 · 실천학습(액션러닝) · 게임기반학습 · 포트폴리오
T
Tied-up (융합)
팀티칭 · 시뮬레이션기반학습(VR) · 융합교육
smart_methods SMART 교육방법 목록 object[]
교육방법명
method_name
string (select)SMART 분류 값 중 선택 — 예) "플립드러닝"
구체적 실행 전략
description
string (textarea)예) 수업 전 영상 강의 시청 → 수업 중 실습·토론 심화
HTML 허용
활용 도구 및 시기

string예) Replit, LMS – 1~7주차

08. 주차별 계획 배열

JSON에서 직접 수정하거나, UI Tab4에서 아코디언을 열어 입력합니다.
weeks 주차별 수업 운영 계획 object[15] 고정

학습목표

→ week_N_goal
